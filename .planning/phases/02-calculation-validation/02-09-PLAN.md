    **RAID 5E/5EE capacity calculations (lines 208-214):**

    Add describe block: "Volumetry Engine - RAID 5E and 5EE Distributed Hot Spare"

    RAID 5E: RAID 5 with integrated distributed hot spare
    - Uses 2 drives worth of overhead (1 parity + 1 distributed spare)
    - Efficiency: (n-2)/n

    RAID 5EE: Similar to 5E but with active hot spare
    - Also uses (n-2)/n efficiency

    Test RAID 5E:
    ```typescript
    it('should calculate RAID 5E capacity correctly (distributed hot spare)', () => {
      const drive = testDrives['hdd-10tb']
      const input: VolumetryInput = {
        topology: { type: 'standard', level: 'RAID5E' },
        driveCount: 8,
        drive,
        serverCount: 1,
        // ... other options
      }
      const result = calculateVolumetry(input)

      // RAID 5E: (8-2)/8 = 75% efficiency (6 data drives, 1 parity, 1 distributed spare)
      expect(result.efficiency).toBeCloseTo(75.0, 1)
      expect(result.usableCapacity).toBeCloseTo(drive.capacity_raw * 6, -9)
    })
    ```

    Tests:
    - RAID 5E with 6 drives (minimum viable)
    - RAID 5E with 8 drives
    - RAID 5E with 12 drives (larger array)
    - RAID 5EE with 6 drives
    - RAID 5EE with 10 drives
