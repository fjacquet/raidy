    **Part B — src/utils/captureChart.ts (new file)**

    ```typescript
    /**
     * Chart capture utilities for PPTX/PDF export.
     * Uses html-to-image to rasterize SVG components into PNG data URLs.
     */
    import { toPng } from 'html-to-image'

    /**
     * Capture the Sankey diagram as a PNG data URL.
     * Returns null if the element is not mounted in the DOM (e.g. collapsed panel).
     */
    export async function captureSankeyDiagram(): Promise&lt;string | null&gt; {
      const el = document.getElementById('sankey-diagram')
      if (!el) return null
      return toPng(el, {
        backgroundColor: '#1A1B2E',
        pixelRatio: 2,
      })
    }
    ```

    - `backgroundColor: '#1A1B2E'` matches the app's surface-900 dark background so the
      captured image blends into the PPTX slide (per EXPORT-06 brand requirement).
    - `pixelRatio: 2` produces a retina-quality image suitable for slide embedding.
    - Keep the function signature exactly as shown — Plan 02 imports it by this name.
